# バス停データ変換ツール

## 概要

国土数値情報（バス停留所データ P11）のGMLファイルをアプリ用のGeoJSON形式に変換するスクリプトです。

---

## 国土数値情報からのダウンロード手順

1. 国土数値情報ダウンロードサービスにアクセスします。
   - <https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-P11.html>
2. 「バス停留所」の宮崎県（コード：46）を選択してダウンロードします。
3. ダウンロードしたZIPファイルを解凍すると、以下のようなファイルが含まれています。
   - `P11-10_46-jgd-g.xml` — バス停の位置・名称・事業者名などを含む本体GMLファイル
   - `KS-META-P11-10_46-jgd-g.xml` — メタデータ（説明書）

---

## convert_bus_stops.py の使い方

標準ライブラリのみで動作するため、追加パッケージのインストールは不要です。

```bash
python tools/convert_bus_stops.py P11-10_46-jgd-g.xml assets/bus_stops/miyazaki_bus_stops.json
```

### 引数

| 引数 | 説明 |
|------|------|
| 第1引数 | 入力GMLファイルのパス（`P11-10_46-jgd-g.xml`） |
| 第2引数 | 出力JSONファイルのパス |

---

## 出力JSONの配置先

変換後のJSONファイルは以下に配置してください。

```
assets/bus_stops/miyazaki_bus_stops.json
```

`pubspec.yaml` にはすでにこのパスがアセットとして宣言されているため、配置後はそのままアプリで読み込まれます。
